# Zoom Plus
![GitHub License](https://img.shields.io/github/license/dodo23333/CUZoomPlus)
![GitHub Release](https://img.shields.io/github/v/release/dodo23333/CUZoomPlus)
![GitHub Downloads (all assets, all releases)](https://img.shields.io/github/downloads/dodo23333/CUZoomPlus/total)

This mod allows you to change the zoom range of the camera. 
If your screen is small, you can make the view larger.
or if you want to see further on high resolution screen or watch further, try this mod!

The mod is compatible with [Casualties Together](https://github.com/creaturefeaturelarry/casualties-together).

## Installation instructions
1. Download BepInEx5
2. Right click the game in steam library and go to the game folder, then unzip the downloaded zip file.
3. Put the mod in Casualties Unknown Demo/BepInEx/plugins
4. Start the game then you can configure the mod(under Video) in game!
5. Enjoy the game!

## Requirements
- [CuCoreLib](https://github.com/jimmyking9999999/CUCoreLib)
